Biblioteca UniFecaf

Projeto de site institucional da Biblioteca UniFecaf, desenvolvido com HTML e CSS, com foco em apresentação da biblioteca, acervo em destaque, unidades e contatos.

Este projeto foi criado para apresentar a Biblioteca UniFecaf em uma interface visual, organizada e responsiva, destacando informações sobre a biblioteca, livros em evidência, unidades e canais de contato.

Funcionalidades
Menu de navegação com versão responsiva em formato hamburger para telas menores.

Seção institucional com apresentação da biblioteca e galeria de imagens.

Grade de livros com organização responsiva para diferentes tamanhos de tela.

Seção de unidades com cards informativos.

Área de contatos com informações principais e redes sociais.

Layout responsivo para celular, tablet, notebook e desktop.

Como executar o projeto
Como este projeto é estático, basta abrir o arquivo HTML principal em um navegador para visualizar o site localmente.

Opção 1: abrir diretamente no navegador
Baixe ou clone este repositório.

Acesse a pasta do projeto.

Abra o arquivo index.html no navegador de sua preferência.

Opção 2: usar o VS Code com Live Server
Abra a pasta do projeto no Visual Studio Code.

Instale a extensão Live Server, se desejar.

Clique com o botão direito no arquivo index.html.

Selecione Open with Live Server.

Responsividade
O projeto foi ajustado para funcionar em diferentes larguras de tela, com adaptação de menu, grids, imagens e espaçamentos.

Os principais cuidados aplicados foram:

Menu adaptado para telas menores.

Reorganização dos cards de livros e unidades conforme o tamanho da tela.

Ajustes em imagens e banners para evitar distorções visuais.

Melhor leitura e navegação em dispositivos móveis.
